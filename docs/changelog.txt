v1.22.16

- improved accuracy of reported video resolution.
- added disk info to notifications
- added a Streamer Setting called "Intro Delay"  This will either delay the recording by X minutes, or split the recording after X minutes.   This allows you to save space by not recording the intro of a stream.  Steps to use:
    1. add the streamer in the Dashboard tab
    2. press enter on the streamer in the Config tab on the STREAMER SETTINGS panel
    3. configure the Intro Delay settings
- fixed issue with the window not responding on some windows 11 machines

- fixed bug with scheduler on fresh installs

- Added a settings menu when you select a streamer in the STREAMER SETTINGS panel.

    1. add the streamer in the Dashboard tab
    2. press enter on the streamer in the Config tab on the STREAMER SETTINGS panel
    3. configure the quality settings
Note: Active recordings must be restarted for the quality settings to take effect. (Disable and then enable the streamer
)

    1. add the streamer in the Dashboard tab
    2. press enter on the streamer in the Config tab on the STREAMER SETTINGS panel
    3. configure the split settings
When recording a stream, the video file will automatically be split into multiple files based on the amount of time you set.
Note: This will override the SPLIT_AFTER site setting.

v1.18.4

- fixed a bug with holding down the arrow keys to scroll which caused 
the scrolling to continue after the keys are released

- added the ability to schedule when a streamer is enabled/disabled.   steps to use:
    1. add the streamer in the Dashboard tab
    2. press enter on the streamer in the Config tab on the PRIORITIES/SCHEDULING panel
    3. configure the schedule
The streamer will be automatically enabled at the start time, and disabled at the end time.

- added twitch ad detection.  steps to use:
    1. enable the the "AD_ALERT" key in the twitch SITE SETTINGS panel
An alert will be displayed in the SYSTEM panel any time an ad is detected in a twitch stream.   
The alert is cleared when the recording ends.  
Note: The ad detection currently only applies to twitch.

- added changelog popup at startup after an update
